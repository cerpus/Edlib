# Contact support

Edlib is a growing open source community.  
On this page we've listed some ways to contact us or how to participate in the discussion.  

Before participating in Edlib's communities, please read our [Code of Conduct](https://github.com/cerpus/Edlib/blob/master/CODE_OF_CONDUCT.md). 

## Discussions

Join our [GitHub Discussions](https://github.com/cerpus/Edlib/discussions) to ask questions and propose features.

## Bugs

Use the GitHub project to report bugs: [Edlib](https://github.com/cerpus/Edlib/issues).

:::note
Please don't use GitHub issues to ask questions.
:::

## News

For the latest news about Edlib, [follow @edlibopensource on Twitter](https://twitter.com/EdlibOpenSource) and our [official Edlib blog](https://docs.edlib.com/blog) on this website.

# H5P

## Requirements
1. Naming: Naming is essential for maintaining compatibility on existing content. The name must be unique in order to avoid conflict with existing h5p packages
2. Format: New H5P libraries must be uploaded as a zipped .h5p package
3. Versioning: Versioning must follow the standards of [semantic versioning](https://semver.org/)

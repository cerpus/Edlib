# Gamilab

## About Gamilab
[Gamilab](https://gamilab.com/) is an online platform where you can find and create easy, fun and educational games. Gamilab uses game elements to motivate and engage players to learn faster and better, and to be more involved in their learning. By using elements like points, stars, highscores and badges Gamilab makes learning feel like a game and motivate the player to learn while playing.

# Generate H5P From QA

## Set Collaborator Context

**Path:** `/common/app/h5p/generate-from-qa`<br/>
**Method:** `POST`<br/>
**Body**

```json
{
  "title": "This is the title", // Required
  "sharing": true, // Required
  "published": true, // Required
  "license": "by", // Required
  "authId": "1", // Required
  "questions": [
    // At least one element is required
    {
      "type": "H5P.MultiChoice", // Required
      "text": "test", // Required
      "answers": [
        // At least one element is required
        {
          "text": "w", // Required
          "correct": false // Required
        },
        {
          "text": "w2", // Required
          "correct": true // Required
        }
      ]
    }
  ]
}
```

# Collaborator Contexts

## Set Collaborator Context

**Path:** `/common/app/context-resource-collaborators`<br />
**Method:** `POST`<br />
**Body**

```json
{
  "context": "context name", // Context name (required)
  "resourceIds": ["resource id"], // A list of resource identifiers (optional)
  "tenantIds": ["user id"], // A list of tenant identifiers. Most likely those are user identifiers (required)
  "externalResources": [
    // An external resource identifier. This should be avoided but is present for compatibility reasons (optional)
    {
      "systemName": "contentAuthor",
      "resourceId": "resource id"
    }
  ]
}
```
